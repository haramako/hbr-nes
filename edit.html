初期設定

Windowsの場合

  1. chocolateyをインストール
  2. (管理者コンソールで) choco install -y ruby cc65-compiler git を実行
  3. (管理者コンソールで) gem install bundle rake を実行
  4. castleフォルダで bundle を実行
  5. これで、build.bat, guard.bat が実行可能になる

Linuxの場合

  1. sudo apt install ruby cc65 libgd3を実行
  2. fc-sampleフォルダで bundle を実行
  3. これで、bundle exec rake (もしくは、bundle2.7 exec rake)

デバッグモード

タイトル画面で「上・上・下・下・左・右・左・右・スタート・セレクト」でデバッグモードON

チェックポイント

  • 種類: "checkpoint" で固定
  • 名前: アルファベット大文字で表示名 (例："CHECKPOINT1")
  • プロパティ: id => 0から連番でチェックポイントの数,

敵

  • 種類: "enemy" で固定
  • 名前: 下表のtypeのいずれかを指定する

敵の種類ごとのプロパティ

※ p1,p2は、省略すると0が使用される

※ 方向は 0:上, 1:右, 2:下, 3:左

※ 速度は単位が書かれていなければ 1/16[px/frame]
